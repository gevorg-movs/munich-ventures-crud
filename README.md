## Installation

```bash
# Install dependencies
$ npm install

# Copy .env.example file and replace the variables with real values
$ cp .env.example .env

# Migrate the database
$ npm run migration:run

# Seed the database
$ npm run seed:run
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
